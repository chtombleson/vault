# Vault

An encrypted document and key value store.

## Depenedencies

SQLCipher

    ubuntu & debain
    $ sudo apt-get install libsqlcipher0 libsqlcipher-dev

## Building

    $ git clone https://github.com/chtombleson/vault.git
    $ cd vault
    $ git submodule update --recursive
    $ make
